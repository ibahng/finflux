API reference
=============

Classes
-------

``finflux`` currently provides four core classes central to its functionality as a financial API:

- ``finflux.equity()`` : Access and analyze individual equity data
- ``finflux.bond()`` : Retrive data on treasury, soverign, and corporate bonds
- ``finflux.indictor()`` : Explore a wide range of economic indicators
- ``finflux.top()`` : Analyze top-performing NYSE/NASDAQ equities across sectors
